# Hands Ready

A browser-based developer toolbox built with React + TypeScript + Vite + Tailwind CSS. All tools run entirely in the browser — no backend required.

## Tools

| Category | Tools |
|----------|-------|
| Development | JSON Formatter, SQL Formatter, Timestamp, MD5, SHA256, Base64, Unicode/Chinese, Byte Calculator, UUID, URL Encode/Decode, Word Count, Byte Count |
| Photography | EXIF Info, Image Compress |
| Others | QR Code, Random Password Generator, Text Diff |

## Reference

HTTP Status Codes, ASCII Table, Time Format Placeholders, HTML Escape Characters, Mirror Sources

## Development

```bash
pnpm install
pnpm dev
```

## Build & Deploy

```bash
pnpm build          # output to dist/
pnpm deploy         # build + deploy to Cloudflare Workers
```

## Tech Stack

React 19 · TypeScript · Vite 7 · Tailwind CSS 4 · React Router · Cloudflare Workers
